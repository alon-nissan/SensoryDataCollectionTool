== VALIDATION TASKS ==

1. MISSED DATA: Scan the article text and tables for any reported sensory measurements that do NOT appear in the extracted results. Focus on:
   - Table cells with numerical data not represented in the results
   - In-text mentions of specific values (e.g., "sweetness was rated 6.2 ± 0.3")
   - Supplementary data referenced in the text
   - Statistical results (p-values, F-statistics) associated with sensory measures
   - Any conditions, time points, or replicates that appear in the paper but are absent from extractions

2. SUSPECT VALUES: Compare extracted values against the original source. Flag any that:
   - Do not match the value in the original table or text
   - Are outside the expected scale range for the experiment method
   - Appear to have decimal point errors (e.g., 82.3 on a 9-point scale)
   - Have mismatched units or scale transformations
   - Have error values larger than the mean

3. CONSISTENCY ISSUES: Check for internal consistency:
   - Do all samples in an experiment have the same set of attributes?
   - Are n values consistent within experiments?
   - Are scale types consistent within experiments?
   - Do sample labels match between experiments?

4. DUPLICATE CANDIDATES: Identify cases where the same underlying data point may have been extracted twice (e.g., once from a table and once from a figure). For each pair:
   - Compare the values — are they identical or within rounding tolerance?
   - Recommend which to keep: table sources are generally more accurate than figures.

== PRIORITY LEVELS ==

- high: Entire table or major data section missed; values clearly wrong by >50%
- medium: Individual values missed or off by 10–50%; minor consistency issues
- low: Rounding differences; optional data like p-values missing; stylistic issues
