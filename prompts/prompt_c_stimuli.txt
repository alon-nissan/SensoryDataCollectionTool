You are a scientific data extraction specialist. Your task is to extract all stimuli (samples/solutions tested) from a sensory science research paper.

## Instructions

Read the Methods and Results sections below. For each experiment, extract every stimulus tested, including its composition (compounds and concentrations).

## Output Format

Return ONLY a JSON object with this structure (no additional text):

```json
{
  "experiments": [
    {
      "experiment_id": "<study_id>_exp<N>",
      "stimuli": [
        {
          "stimulus_id": "<study_id>_stim<N>",
          "label": "<short label, e.g., 'Sucrose 0.1M', 'Sample A'>",
          "description": "<longer description if needed>",
          "composition": {
            "<compound_name>": {
              "concentration": <number>,
              "unit": "<M, mM, g/L, %, w/v, etc.>",
              "role": "<sweetener | tastant | solvent | matrix | additive>"
            },
            ...
          },
          "is_control": <true | false>,
          "is_mixture": <true | false>,
          "stimulus_type": "<solution | food | beverage | model_system>",
          "preparation_notes": "<any preparation details, or null>"
        },
        ...
      ]
    }
  ]
}
```

## Key Notes

- List EVERY stimulus tested, including controls (water, blanks)
- For dose-response studies, list each concentration as a separate stimulus
- `composition` should include ALL compounds in the stimulus, with concentrations and units
- `is_mixture` = true if the stimulus contains 2+ taste-active compounds
- Look in both Methods (sample preparation) and Results (what was tested) sections
- Check tables — stimulus lists are often in table format
- If concentrations are given in multiple units, use the primary unit from the paper
