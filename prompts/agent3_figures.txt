== INSTRUCTIONS ==

1. Examine the figure carefully and identify every numerical data point visible.
2. PREFER using existing sample_ids from the provided list. Match by substance name, concentration, and any other identifying info in the figure legend/labels. If you encounter a sample in the figure that clearly does not match ANY existing sample, you MAY create a new sample_id using the format "{paper_id}__sample_fig_{N}" with source_type: "figure_created" and include full identifying info (substance name, concentration, matrix) in the result's context_json field. 3. Skip values already present in existing_results_summary to avoid duplication. If a value for the same sample × attribute combination already exists from a table or a prior figure, do NOT re-extract it. Record skipped values in "skipped_values".
4. For each NEW data point, create a result row with all required fields.

== READING FIGURES ==

- Bar charts: Read the bar height against the y-axis. Read error bars (top of bar to top of error bar = error value). Note whether error bars are SE or SD (check the caption/legend).
- Line plots: Read each data point where a marker appears. Interpolate from gridlines.
- Spider/radar plots: Read each spoke value for each sample.
- Dose-response curves: Read each plotted point along the curve.
- Box plots: Extract median, Q1, Q3, whisker endpoints if readable.
- PCA/MFA biplots: Do NOT extract coordinates as sensory data — note in extraction_notes if relevant loadings are visible.
- Heatmaps: Read cell values if labeled; otherwise estimate from the color scale.

== CONFIDENCE RATING ==

Rate extraction_confidence for each value:
- "high": Value readable within ±2% of scale range, clear gridlines or labeled data points.
- "medium": Value estimated within ±5–10% of scale range, interpolated between gridlines.
- "low": Value estimated with >10% uncertainty, overlapping elements, poor resolution, or ambiguous reading.
